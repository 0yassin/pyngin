### Alternatively you can run it locally
- step 1: clone the repo:
    ```git clone https://github.com/0yassin/pyngin.git```
- step 2: cd to the folder: ```cd pyngin/pygame-executable```
- step 3: install dependancies:
     ```pip install requirements.txt```
- step 4: run the program: ```python main.py```
